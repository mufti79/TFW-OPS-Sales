# Public Assets Directory

This directory contains static assets for the TFW OPS & Sales application.

## Logo

The `logo.svg` file is the default application logo. To customize it:

1. **Via the Application Interface** (Recommended):
   - Log in as an admin
   - Click on the "Backup" button in the header
   - In the Backup Manager modal, scroll to the "Manage Application Logo" section
   - Click "Choose Image..." to upload your logo (PNG, JPEG, or WebP)
   - The logo will be automatically resized to 256x256 pixels
   - Click "Save New Logo" to apply it

2. **Manually Replace the File**:
   - Replace `logo.svg` with your own logo file
   - Supported formats: SVG, PNG, JPG
   - Recommended size: 200x200 pixels minimum
   - The logo will appear in:
     - Application header (top left)
     - Login screen
     - Browser tab icon (favicon)

## Adding More Assets

You can add more static assets to this directory:
- Ride images
- Icons
- Backgrounds
- Other graphics

These assets will be available at `/filename.ext` in your application.
